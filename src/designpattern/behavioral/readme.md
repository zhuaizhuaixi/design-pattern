# 行为型软件设计模式

行为型设计模式关心算法和对象之间的责任分配，关心的不只是描述对象或类的模式，而更侧重他们之间的通信模式。

行为型软件设计模式解释对象之间是如何进行交互的。

介绍/学习的种类：
1. 迭代器模式 Iterator
2. 访问者模式 Visitor
3. 命令模式 Command
4. 中介者模式 Mediator
5. 策略模式 Strategy
6. 状态模式 State
7. 解释器模式 Interpreter
8. 备忘录模式 Memento
9. 模板方法模式 Template Method
10. 责任链模式 Chain of Responsibility
11. 观察者模式 Observer
